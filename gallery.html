I have designed and implemented a fully responsive Gallery section for the project. The layout is built using CSS Grid To ensure a clean and structured design across all screen sizes. The gallery displays five images per row on desktop, four on laptops, three on tablets, and one image per row on mobile devices between 320px and 480px. All images are set to a uniform size using fixed height and object-fit properties to maintain visual consistency. Smooth hover effects and transition animations have been added to enhance user interaction and provide a modern, professional look. The section is optimized for responsiveness, performance, and clean UI presentation.
